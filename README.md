spring-docker
=============

Spring-bootを使った簡単なTodoList  

Usage:
------

レポジトリをclone  

```
git clone git@github.com:ymzkryo/spring-docker.git
cd spring-docker
```

docker-composeでMysqlを起動  

```
docker-compose up -d --build
```

spring-bootを実行し、`localhost:8080`にアクセス  


Requirements:
-------------

- Java11
- mvn
- docker
- docker-compose
